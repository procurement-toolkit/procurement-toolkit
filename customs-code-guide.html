해외직구 가이드

개인통관고유부호 발급 방법, 3분이면 끝나요

해외직구 할 때 꼭 필요한 개인통관고유부호. 2026년 2월부터 인증 방식이 바뀌었는데, 이 글에서 최신 기준으로 발급부터 갱신까지 한 번에 정리했습니다.

⚠ 2026년 2월 2일부터 바뀐 것 (꼭 확인하세요)

기존에는 이름 + 전화번호만 맞으면 통관이 됐지만, 이제는 여기에 배송지 우편번호까지 일치해야 통관이 진행됩니다. 그리고 개인통관고유부호에 유효기간 1년이 새로 생겨서, 발급받고 1년이 지나면 다시 갱신해야 합니다. 예전에 발급받아 두신 분도 이 글 하단의 "갱신 확인" 부분을 꼭 확인해주세요.

개인통관고유부호가 뭔가요?

해외에서 물건을 사서 국내로 들여올 때, 세관이 "이 물건은 누가 산 거다"를 확인하기 위해 필요한 개인 식별 번호입니다. 예전에는 주민등록번호를 그대로 썼는데, 개인정보 유출 위험 때문에 지금은 P로 시작하는 13자리 전용 번호를 대신 사용합니다.

이 번호가 없거나 틀리면 물건이 세관에 묶여서 통관이 안 되고, 배송이 며칠씩 지연될 수 있어요. 아마존, 알리익스프레스 등에서 해외 배송으로 직접 주문할 때 결제 단계에서 이 번호를 입력하라고 나오는 경우가 많습니다.

발급 방법 (3분 완성)

관세청 유니패스 접속

아래 버튼으로 유니패스(unipass.customs.go.kr) 개인통관고유부호 발급 페이지로 이동하세요.

본인인증

휴대폰 인증 또는 공동인증서(구 공인인증서)로 본인 확인을 합니다. 휴대폰 인증이 가장 빠릅니다.

영문 이름 + 배송지 우편번호 등록

여권에 표기된 영문 이름과, 자주 받는 배송지 우편번호를 함께 등록하세요. 2026년부터 이 정보가 통관 검증에 쓰이기 때문에, 정확히 입력하는 게 중요합니다.

번호 발급 완료

인증이 끝나면 화면에 P로 시작하는 13자리 번호가 바로 나옵니다. 스마트폰 메모장 등에 저장해두면 다음 직구 때 편해요.

관세청 유니패스에서 발급받기 →
발급은 무료이며, 한 번 발급받으면 여러 쇼핑몰에서 같은 번호를 반복해서 쓸 수 있습니다 (단, 2026년부터는 1년마다 갱신 필요).
이미 번호가 있는데 통관이 안 돼요 — 갱신·재발급 확인

유효기간이 지났을 수 있어요. 2026년부터 도입된 1년 유효기간이 지나면 번호가 자동으로 만료됩니다. 유니패스에서 "개인통관고유부호 조회"로 유효 여부를 확인하고, 만료됐다면 같은 절차로 재발급받으면 됩니다.

배송지 우편번호가 안 맞을 수도 있어요. 등록해둔 배송지와 실제 주문 시 입력한 배송지의 우편번호가 다르면 통관이 지연됩니다. 새 주소로 배송받는다면, 유니패스에서 배송지를 먼저 추가 등록해두세요 (최대 20건까지 등록 가능).

도용이 의심된다면 유니패스의 "개인통관고유부호 도용신고" 메뉴에서 본인인증 후 바로 신고하고 번호를 재발급받을 수 있습니다.

자주 묻는 질문
발급 비용이 드나요?

아니요, 무료입니다. 관세청이 직접 운영하는 공식 서비스라 어떤 비용도 들지 않습니다.

구매대행을 이용하면 이 번호가 필요 없나요?

구매대행 업체가 대신 통관을 진행하는 경우, 업체가 자체 번호로 처리하기도 하지만 본인 명의 번호를 요구하는 곳도 많습니다. 업체 안내를 따르시되, 미리 발급해두면 어디서든 바로 쓸 수 있어 편합니다.

가족 것도 하나씩 따로 발급받아야 하나요?

네, 개인마다 본인 명의로 각각 발급받아야 합니다. 타인에게 양도하거나 대여하는 것은 금지되어 있습니다.

번호를 잊어버렸어요.

유니패스나 모바일관세청 앱에서 본인인증 후 "조회" 메뉴로 바로 확인할 수 있습니다. 새로 발급받을 필요 없이 기존 번호를 다시 볼 수 있어요.
