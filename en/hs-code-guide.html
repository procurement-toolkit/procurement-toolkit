How HS Code Structure Works

An HS code consists of a 6-digit international code set by the World Customs Organization (WCO), with each country adding further digits for its own statistical and tariff purposes.

Example: 8471.30 (portable computers, the first 6 digits are globally standardized)
→ The US extends this to a 10-digit HTS code
→ The EU uses an 8-digit CN (Combined Nomenclature) code
→ South Korea uses a 10-digit HSK (HS of Korea) code

So matching the first 6 digits gets you the same broad international category, but you need the destination country's full-length code to determine the exact applicable duty rate.

3 Steps to a More Accurate Search

Search with product + material + function

Instead of just "bag," search "leather handbag" or "polyester backpack" — including material and intended use narrows the results significantly.

Cross-check with past classification cases

Many official portals (like Korea's CLIP) let you search how similar products were actually classified in past declarations — checking this improves accuracy.

Request a binding ruling if it's ambiguous

For genuinely ambiguous products, most customs authorities (like CBP's ruling requests, or Korea's pre-classification review) let you request an official, binding classification determination — usually with a fee and processing time.

Common Mistakes
Misclassifying by intended use

The same "glass cup" can fall under a different code depending on whether it's decorative or for kitchen use.

Misjudging material composition

Thinking a bag is genuine leather when it's actually synthetic (or vice versa) can change the duty rate enough to trigger a retroactive assessment.

Not verifying the tariff-shift rule for FTA claims

Claiming a preferential FTA rate requires the HS code to actually shift as a result of processing — skipping this check is a common reason exporters get flagged in a post-entry audit.

Frequently Asked Questions
What happens if I misclassify an HS code?

You could overpay or underpay duty, face customs delays, or get flagged in a post-entry audit with penalties and retroactive assessment. For ambiguous products, requesting a binding ruling in advance is the safer path.

Are export codes different from import codes?

Yes, this can vary by country. The US, for example, uses a 10-digit HTS code for imports and a separate 10-digit Schedule B code for exports, though the first 6 digits are identical.

Does the duty rate differ by country even for the same HS code?

Yes. Even with matching first-6-digit codes, the actual applicable duty rate can vary significantly depending on the importing country, country of origin, and whether an FTA applies. Always check the destination country's own tariff schedule separately.
