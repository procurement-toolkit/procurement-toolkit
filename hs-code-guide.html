HS Code 조회 & 관세율 확인 가이드

HS Code를 잘못 찾으면 관세 과오납, 통관 지연, 심하면 추징까지 이어질 수 있습니다. 저희가 직접 코드나 세율을 알려드리는 대신, 각국 관세당국의 공식 조회 도구로 바로 연결해드리고, 정확하게 검색하는 요령을 정리했습니다.

품목분류공식 데이터 연결관세율 조회
⚠ HS Code 분류는 전문 분류업체의 AI 도구조차 정확도가 70~80%대에 머무를 만큼 까다로운 영역입니다. 이 페이지는 직접 코드를 판정해드리지 않고, 공식 관세당국 데이터로 바로 연결해드립니다. 고액 거래나 중요한 판정은 반드시 관세사와 상담하세요.

HS Code, 이렇게 구성됩니다

HS Code는 세계관세기구(WCO)가 정한 앞 6자리 국제 공통 코드에, 각 나라가 자국 통계·관세 목적으로 자릿수를 추가하는 구조입니다.

예: 8471.30 (노트북 컴퓨터, 앞 6자리는 전 세계 공통)
→ 미국은 여기에 4자리를 더해 10자리 HTS 코드로 확장
→ EU는 8자리 CN(Combined Nomenclature) 코드 사용
→ 한국은 10자리 HSK(HS of Korea) 코드 사용

즉 앞 6자리만 맞으면 국제적으로 같은 대분류지만, 정확한 세율 적용을 위해서는 수입국 기준의 전체 자릿수까지 확인해야 합니다.

정확하게 검색하는 3단계

제품명 + 재질 + 용도로 검색

단순히 "가방"이 아니라 "가죽 핸드백" 또는 "폴리에스터 백팩"처럼 재질과 용도를 함께 넣으면 검색 범위가 좁혀집니다.

국내사례 검색으로 교차 확인

유니패스 CLIP의 "국내사례" 메뉴에서 비슷한 물품이 실제로 어떤 코드로 신고됐는지 과거 사례를 확인하면 정확도가 올라갑니다.

애매하면 관세평가분류원에 사전심사 요청

애매한 품목은 관세청 관세평가분류원에 품목분류 사전심사를 신청하면 공식적으로 확정된 코드를 받을 수 있습니다 (수수료 있음, 처리에 시간 소요).

자주 하는 실수
용도에 따른 분류 오류

같은 "유리컵"이라도 장식용인지 주방용인지에 따라 코드가 달라질 수 있습니다.

재질 판단 착오

가죽 가방인 줄 알았는데 합성피혁이었던 경우처럼, 재질 하나 차이로 관세율이 달라져 추징 대상이 될 수 있습니다.

HS Code를 잘못 신고하면 어떻게 되나요?

관세 과오납(더 내거나 덜 내거나), 통관 지연, 심한 경우 세관 사후 심사에서 적발되어 가산세와 함께 추징될 수 있습니다. 애매한 품목은 사전심사를 받는 것이 안전합니다.

수출용 코드와 수입용 코드가 다른가요?

네, 나라에 따라 다를 수 있습니다. 예를 들어 미국은 수입 시 10자리 HTS 코드, 수출 시에는 별도의 Schedule B 코드(역시 10자리)를 사용하며, 앞 6자리는 서로 같습니다.

같은 HS Code라도 나라마다 관세율이 다른가요?

네, HS Code 앞 6자리가 같아도 실제 적용되는 관세율은 수입국, 원산지, FTA 체결 여부에 따라 크게 달라질 수 있습니다. 코드 확인 후 반드시 수입국 기준 관세율표를 따로 확인해야 합니다.
